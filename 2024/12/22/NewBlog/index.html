使用 GitHub 和 Hexo 搭建博客方案

介绍

Hexo 是一个快速、简洁且高效的静态博客框架，支持 Markdown 格式的文章撰写。结合 GitHub Pages，您可以轻松地将博客托管在 GitHub 上，享受免费的托管服务。

环境准备

在开始之前，请确保您已经安装了以下工具：

  • Node.js（推荐使用 LTS 版本）
  • Git
  • Hexo

安装 Hexo

  1. 全局安装 Hexo CLI

    打开终端，运行以下命令：

    npm install -g hexo-cli
